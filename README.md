# History Journeys - Indian Travel Guide

Welcome to the **History Journeys** project! This is a responsive, front-end web application designed to showcase Indian heritage, tourism, and travel packages. The website invites users to "explore our nation, discover new places, and feel the essence" of India.

## 🚀 Features

- **Responsive Design**: The website is fully responsive, featuring a hamburger menu for smooth navigation on mobile and tablet devices.
- **Dynamic Video Background**: The home section features an interactive video slider with controls, allowing users to switch between different beautiful video clips of Indian destinations.
- **Login Modal**: An integrated, interactive login and registration modal popup that can be accessed from the navigation bar.
- **Search Functionality**: A sleek, toggleable search bar interface in the header.
- **Multi-page Structure**: Organized into separate HTML pages including Home, Booking, Packages, Services, Gallery, Reviews, and Contact.
- **Interactive UI Elements**: Smooth scrolling, hover effects, and active state styling.
- **Travel Packages**: A dedicated section displaying various travel destinations (e.g., Mumbai, Hyderabad, Gujarat, Delhi) complete with pricing, star ratings, and booking options.

## 🛠️ Tech Stack

This project is built purely with core web technologies, ensuring fast load times and no external framework dependencies:

- **HTML5**: For structuring the content and multiple pages.
- **CSS3**: For styling, responsive layouts (Flexbox/Grid), and animations.
- **Vanilla JavaScript**: For DOM manipulation, interactive elements (video slider, login modal, menu toggles), and event handling.
- **FontAwesome (v5.15.3)**: Utilized for high-quality, scalable icons throughout the website (search, user login, menu, location pins, and star ratings).

## ⚙️ How to Run Locally

1. Clone or download the repository to your local machine.
2. Open `index.html` in your preferred web browser to view the application. No build steps or local servers are strictly required!
